Api Documentation

World Information API

An API for world information.

Introduction

World Information API provides information about continents, countries and cities in the world. It also provides total population in particular country and city.

Access Restrictions

All endpoints are free and publicly accessible to everyone from everywhere.

Requests

All endpoints take only GET HTTP requests, with all arguments given as path parameters (in the URL).

Responses

The response format depends on the 'Accept' header.

The standard response format is JSON (application/json) with CORS for cross-domain requests.

Request Parameters

Certain parameters are required while others are optional. As is standard in URLs, all parameters are separated using the ampersand (&) character. The list of parameters and their possible values are enumerated below.


Required Parameter

Action- This represents what information is requested.

Request Structure -Continent List

A sample HTTP request is shown below, getting list of Continents

http://worldinform.azurewebsites.net/api/continents

Response Structure

JSON Output

The JSON result is shown below.

{ "Version": "1.0", "StatusCode": 200, "Result": [ { "Name": "Africa", "Countrycount": 58 }, { "Name": "Antarctica", "Countrycount": 5 }, { "Name": "Asia", "Countrycount": 51 }, . . ] }

Country List

Gets the list of all countries with total Population, Region etc.

http://worldinform.azurewebsites.net/api/countries

JSON Output

The JSON result is shown below.

{ "Version": "1.0", "StatusCode": 200, "Result": [ { "ID": "ABW", "Name": "Aruba ", "Continent": "North America", "Region": "Caribbean ", "SurfaceArea": 193, "IndepYear": null, "Populationn": 103000, "LifeExpectancy": 78.4, "GNP": 828, "GNPOld": 793, "LocalName": "Aruba ", "GovernmentForm": "Nonmetropolitan Territory of The Netherlands ", "HeadOfState": "Beatrix ", "Capital": 129, "Code2": "AW", },.. ] }

City List

Gets the list of all cities with total Population, Country etc.

http://worldinform.azurewebsites.net/api/cities

JSON Output

The JSON result is shown below.

{ "Version": "1.0", "StatusCode": 200, "Result": [ { "ID": 1, "Name": "Kabul ", "CountryCode": "AFG", "District": "Kabol ", "Populationn": 1780000, "country": null },..]}

About

World Information API is an open source project which provides information regarding Continents, Countries and Cities in World.
